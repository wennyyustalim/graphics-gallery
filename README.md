# Installation Guide

1. Clone project repository from terminal

```
git clone https://github.com/johnLucious/mouse_input.git
```

2. Compile project

```
make
```

3. Switch to TTY

4. Run the program as superuser

```
sudo make run
```

5. Enjoy!
